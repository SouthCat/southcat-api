  • 一言api
  • 请求地址:https://api.southcat.net/yiyan
  • 请求方式:GET
  • 请求编码:charset 参数:UFT8/GBK
  • 请求参数:接口:code参数:js
  • 调用举例 – JavaScript + HTML
    <script type="text/javascript" src="https://api.southcat.net/yiyan/?code=js"></script><div id="southcat"><script>southcat()</script></div>
  • IP查询API
  • 请求地址:https://api.southcat.net/ip
  • 请求方式:GET
  • 请求参数:接口:code
  • 参数:1.js(返回:你的ip是:xxx)
  • 2.js-txt(只返回ip)
  • 调用举例-JavaScript+HTML
    <script type="text/javascript" src="https://api.southcat.net/ip/?code=js"></script><script>scip()</script>
